## 🔨 Funcionalidades do projeto

Cronômetro que permite iniciar, pausar, prosseguir e resetar a contagem de Minutos/Segundos/MiliSegundos.

## ✔️ Tecnologias utilizadas

O Projeto utilizou HTML, CSS e JavaScript.
